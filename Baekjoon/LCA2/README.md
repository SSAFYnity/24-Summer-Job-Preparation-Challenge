## Code Review
1. 같은 언어를 풀이하는 참가자들에 최대 14명 랜덤하게 리뷰어로 배정됩니다.
2. 참가자 리뷰 외 운영진 리뷰가 필요하면 "Pull Request" 사이드 바의 "Revewier"에 `@eona1301`을 추가하세요.
3. 한 Pull Request 당 최소 3명의 Approve가 필요합니다.
4. 서로의 코드를 확인하고 인사이트를 받으며, 다양한 풀이를 확인해보세요.
